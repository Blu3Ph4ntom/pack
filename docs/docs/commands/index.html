Commands

These command groups come directly from the clap subcommands defined in crates/pack-cli/src/main.rs.

Gem-compatible verbs

pack install rails

Install a specific gem through the native manager path.

pack list rails

List installed gems, optionally matching a pattern.

pack search rails

Search remote gems.

pack info rails

Inspect gem metadata and dependencies.

pack env

Show gem-related environment information.

pack outdated

Check for outdated gems.

Bundler-adjacent flows

pack install

Runs a Gemfile install path when no specific gem is supplied.

pack exec rails console

Run gem binaries directly with a “faster front door” posture.

pack add rails

Add a dependency to the Gemfile.

pack remove rails

Remove a dependency from the Gemfile.

pack update

Update all or one gem entry.

pack generate

Generate or update Gemfile.lock.

pack why rack

Trace why a gem appears in the dependency tree.

pack lock

Write a binary pack.lock.

Tasks and diagnostics

pack doctor

Inspect project state, Ruby environment, cache directories, gems, plugins, and Rails issues.

pack run dev

Run a named task from a Packfile.

pack tasks

List Packfile task names and optionally descriptions.

pack plugins list

Inspect installed Pack plugins.

Rails shortcuts

pack server

Start the Rails server.

pack console

Open Rails console.

pack test

Run Rails tests.

pack rspec

Run RSpec with passthrough args.

pack db migrate

Run database operations.

pack assets precompile

Drive asset pipeline operations.

pack cache clear

Run cache helpers.

pack docker

Generate Docker support for a Rails app.
