Features

Everything you need. Nothing you don't.

A modern Ruby package manager built for speed and developer experience.

⚡

Native Speed

Written in Rust with zero-copy parsing. No Ruby interpreter startup overhead means sub-millisecond operations.

📦

Native Gems

Install gems directly from RubyGems.org API. No bundler, no gem binary required. Pack works on any system.

🔒

pack.lock

Binary MessagePack format that's 100x faster than Gemfile.lock. Parse 200 gems in 41 microseconds.

🐳

Docker Ready

No Ruby required for gem operations. Smaller Docker images, faster builds, no gem path issues.

🎯

Rails Native

First-class Rails support with commands like pack server, pack console, pack db migrate.

🔌

Plugin System

Extend pack with plugins. Shell scripts, binaries, Docker containers — any executable.

📝

Packfile Tasks

Define project tasks in a Packfile. pack run dev, pack run deploy, pack run test.

🏃

Offline Mode

Use cached gems without network. Perfect for CI/CD pipelines and air-gapped environments.

First-Class Support

Made for Rails

Pack understands your Rails project. First-class commands for server, console, tests, database, assets, and more. Works perfectly with Rails 8, Solid Queue, Solid Cache, and Propshaft.

Get Started

Start in 30 seconds

1

Install Pack

Download and install the pack binary. Works on Linux, macOS, and Windows.

2

Create a Rails project

Use pack new to scaffold a new Rails app with Docker support, or initialize an existing project.

3

Install gems and go

Pack install is faster than bundle install. Generate pack.lock for near-instant parsing.

terminal
# Install pack
$ curl -fsSL https://pack.dev/install.sh | bash

# Create new Rails project
$ pack new myapp --docker --database postgresql

# Go into the project
$ cd myapp

# Install gems (faster than bundle!)
$ pack install

# Start server
$ pack server
