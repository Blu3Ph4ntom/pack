Ruby Package Manager

Pack is an ultra-fast Ruby package manager written in Rust. Sub-second dependency resolution, native gem installation, and first-class Rails support. No Ruby required.

Get Started View Benchmarks
curl -fsSL https://pack.dev/install.sh | bash

Why Pack?

⚡

100x Faster

Lockfile parsing is 33,000x faster than Bundler. Native Rust parsing means no Ruby interpreter startup overhead.

🔧

No Ruby Required

Works on systems without Ruby installed. Perfect for Docker containers and CI/CD pipelines.

🚂

Rails First

First-class Rails support with commands like pack server, pack console, pack test, and pack db.

📦

Binary pack.lock

MessagePack-based lockfile format for near-instant parsing. 100x faster than Gemfile.lock.

🔌

Plugin System

Extend Pack with plugins. Shell scripts, binaries, or Docker containers — any executable works.

📦

Offline Mode

Cached gems work without network. Set PACK_OFFLINE=1 and keep building.

Get Started

1

Install Pack

curl -fsSL https://pack.dev/install.sh | bash
2

Initialize Project

cd my-rails-app
pack init
3

Install Gems

pack install

Commands

pack install Install gems from Gemfile
pack add rails Add gem to Gemfile
pack remove rails Remove gem from Gemfile
pack update Update gem versions
pack why rails Explain dependency chain
pack gem list List installed gems
pack server Start Rails server
pack console Open Rails console
